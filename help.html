URL Functions List

  • /getAllEmp - Getting All Employees
  • /getAllMem - Getting All Library Members
  • /getAllBooks - Getting All Books In The Library
  • /oldestMember - Getting The Oldest Member(Joined To The Library First
  • /oldestEmp - Getting The Oldest Employee
  • /bestSeller - Getting The Top Seller
  • /BooksByID/:bookID - Getting A Book By ID
  • /BooksByName/:bookName - Getting A Book By Name, Put The Name Without: ""
  • /find?price=xxx - Getting A Book That Its Price Is "xxx"
  • /find?title=yyy - Getting A Book That Its Title Is "yyy"
  • /find?price=xxx&title=yyy - Getting A Book That Its Price Is "xxx" And Its Title Is "yyy"
